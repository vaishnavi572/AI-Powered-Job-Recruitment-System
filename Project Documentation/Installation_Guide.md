# Installation Guide

## Prerequisites

- Salesforce Developer Edition
- Salesforce CLI
- Visual Studio Code
- Salesforce Extension Pack

---

## Installation Steps

1. Clone the repository.

2. Open the project in Visual Studio Code.

3. Authenticate with Salesforce.

4. Deploy the metadata to the target org.

5. Assign the required permissions.

6. Open the Lightning App.

---

## Verification

After deployment verify:

- Custom Objects
- Flows
- Approval Processes
- Validation Rules
- Agentforce AI
- Reports
- Dashboards
