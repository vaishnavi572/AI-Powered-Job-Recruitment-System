# Project Demonstration

## Overview

The Project Demonstration phase showcases the implementation and functionality of the AI-Powered Job Recruitment & Candidate Management System (AIRCMS). It highlights the application's key features, workflows, automation, and AI capabilities developed using Salesforce.

## Contents

- ProjectDemonstration.md
- DemoVideoLink.md

## Purpose

The purpose of this phase is to demonstrate how the system operates in real-time, illustrating the complete recruitment workflow from candidate application to hiring decisions using Salesforce automation and Agentforce AI.
