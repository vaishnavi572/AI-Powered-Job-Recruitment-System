# Project Development Phase

## Overview

The Project Development Phase focuses on implementing the AI-Powered Job Recruitment & Candidate Management System (AIRCMS) using Salesforce.

The application was developed using Salesforce Lightning Experience, Custom Objects, Flows, Approval Processes, Validation Rules, Reports, Dashboards, and Agentforce AI to automate and optimize the recruitment lifecycle.
